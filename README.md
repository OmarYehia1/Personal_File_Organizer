# Personal File Organizer Workspace

A hands-on Linux terminal project built entirely from the command line — no GUI.

## What I built
A structured directory workspace to practice core Linux CLI skills as part of my DevOps learning journey.

## Commands used
- `mkdir` — created the directory structure
- `touch`, `mv`, `cp` — created and organized files
- `chmod`, `chown` — applied file permissions to secure directories
- `vi` — edited files directly from the terminal
- Error redirection (`2>`) — redirected error output to a log file

## Why I built it
To get hands-on with the Linux command line before moving into DevOps tools. Everything here was done without a GUI to build real terminal confidence.
